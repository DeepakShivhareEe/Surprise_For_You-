## 🎯 VERSION A: HTML/CSS/JS (Instant Start)

### ⚡ FASTEST WAY (10 seconds)
1. Open folder: `version-a/`
2. Double-click: `index.html`
3. Done! 🎉

### ✏️ CUSTOMIZE (2 minutes)
1. Open `script.js` in any text editor
2. Find the `CONFIG` section (lines 10-30)
3. Edit:
   ```javascript
   nickname: 'YourNickname',
   favoriteColor: '#FF6B9D',
   insideJokeLine: 'Your inside joke here',
   emotionalMessages: ['Message 1', 'Message 2', ...],
   songFileURL: 'your-music-url.mp3'
   ```
4. Save and refresh browser

### 🎵 ADD MUSIC (optional)
- Place `music.mp3` in `version-a/` folder
- Set: `songFileURL: './music.mp3'`

## 🚀 VERSION B: React + Framer Motion (Professional)

### 📦 INSTALL (1 minute)
```bash
cd version-b
npm install
```

### ▶️ RUN (1 click)
```bash
npm run dev
```
Opens at: http://localhost:3000

### ✏️ CUSTOMIZE (2 minutes)
1. Open `src/config.js`
2. Edit the CONFIG object
3. Save (auto-reloads!)

### 🏗️ BUILD FOR PRODUCTION
```bash
npm run build
```
Deploy the `dist/` folder anywhere!

## 🐛 QUICK TROUBLESHOOTING

### Music Not Playing?
→ Click the music toggle button (🎵)  
→ Check browser console for errors  
→ Try using http://localhost server

### Changes Not Showing?
→ Hard refresh: `Ctrl + Shift + R` (Windows) or `Cmd + Shift + R` (Mac)  
→ Clear browser cache

### Version B Won't Start?
→ Make sure Node.js is installed  
→ Delete `node_modules/` and run `npm install` again
